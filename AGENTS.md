# AGENTS

- Используйте русский язык в описаниях и комментариях, если иное не требуется.
- Заголовки Markdown пишите в sentence case и делайте информативными.
- Форматируйте текстовые файлы с переносами строк до ~120 символов.
- Перед изменениями изучайте текущее поведение и сохраняйте уже принятый стиль кода и шаблонов.
- Если добавляете Python-файлы, обеспечьте наличие понятных docstring'ов и типовых подсказок там, где это
  упрощает чтение.
- Требования в `requirements*.txt` держите отсортированными по алфавиту и с минимально необходимыми
  версиями.
- По возможности запускайте `python -m unittest discover` перед коммитом и фиксируйте результат в описании
  работы.
- Для README и прочей документации добавляйте короткие инструкции по запуску и тестированию, чтобы новички
  могли быстро включиться в работу.
- В этом репозитории вместе с UsersDash лежат Windows-сборки `RSSv7` и `RSS4SALE`. При правках не ломайте
  пути для Windows и помечайте, если нужен запуск от администратора или конкретные версии Python.
- Описывайте изменения в README/AGENTS при добавлении новых утилит, конфигов или шаблонов, особенно в
  каталогах `UsersDash/bot_farm_configs`, `RSSv7` и `RSS4SALE`.
